# Welcome to the Start Games Open Source Project's GitHub Organization!

Currently its barebones but you can see our open source projects repositories that you can fork, make pull requests, etc.

### Contact Us
If you want to reach out, you can create a [github discussion](https://github.com/orgs/Start-Games-Open-Source/discussions) or an issue on the corresponding repository so we can track anything that needs to be done or discussed.
Please respect each member's privacy and go through those public channels for communication. Social media messaging has some leaway but again be respectful of people's time and availability.
